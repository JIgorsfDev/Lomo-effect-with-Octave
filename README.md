# 📷 Lomo Effect with GNU Octave

## Image Processing Practical Assignment

This repository contains the submission of the practical **Image Processing** assignment developed for the **Computer Graphics** course during the 5th semester of the **Bachelor's Degree in Computer Science** at the Federal Institute of Education, Science and Technology of Ceará (IFCE) – Tianguá Campus, taught by **Prof. Dr. Nécio de Lima Veras**. 

The objective of the assignment was to implement one of the image processing topics proposed during the course using **GNU Octave**. The selected topic was the implementation of the **Lomo Effect**, applying digital image processing techniques to reproduce its characteristic visual style.

---

## Trabalho Prático de Processamento de Imagens

Este repositório refere-se à entrega do trabalho prático de **Processamento de Imagens**, desenvolvido na disciplina de **Computação Gráfica** do 5º semestre do curso de **Bacharelado em Ciência da Computação** do Instituto Federal de Educação, Ciência e Tecnologia do Ceará (IFCE) – Campus Tianguá, ministrada pelo **Prof. Dr. Nécio de Lima Veras**.

O trabalho teve como objetivo implementar um dos temas propostos durante a disciplina utilizando o **GNU Octave**. O tema escolhido foi a criação de um filtro baseado no **Efeito Lomo**, aplicando técnicas de processamento de imagens para reproduzir suas características visuais.

---

## ✨ Features | Funcionalidades


- 📷 Applies the Lomo effect to digital images.
- 🎨 Enhances image colors and contrast.
- 🌑 Adds a vignette effect.
- 🖼️ Demonstrates digital image processing techniques using GNU Octave.

- 📷 Aplicação do efeito Lomo em imagens digitais.
- 🎨 Ajuste de cores e contraste.
- 🌑 Aplicação do efeito de vinheta.
- 🖼️ Demonstração de técnicas de processamento digital de imagens utilizando GNU Octave.
